# 모바일 반응형 체크리스트

이 문서는 **모바일 화면에서 UI가 잘 줄어드는지 확인하기 위한 체크리스트**입니다.

---

## 1) 화면 크기 설정

1. 브라우저 개발자 도구 열기 (F12)
2. 모바일 보기(Toggle Device Toolbar) 활성화
3. 폭 375px ~ 390px 정도로 설정

추가: 태블릿 확인도 권장합니다.
- 폭 768px 정도로 설정

---

## 2) 공통 확인 항목

- [ ] 글자 크기가 너무 작거나 크지 않은가
- [ ] 버튼이 화면 밖으로 잘리지 않는가
- [ ] 입력 폼이 화면을 넘치지 않는가
- [ ] 여백이 너무 크거나 너무 작지 않은가
- [ ] 스크롤이 이상하게 생기지 않는가

---

## 3) Week04 체크

- [ ] 리스트 항목이 줄바꿈 없이 보이는가
- [ ] 목록 간 간격이 너무 넓지 않은가

---

## 4) Week05 체크

- [ ] 햄버거 버튼이 보이는가
- [ ] 햄버거 버튼 클릭 시 사이드바가 열리는가
- [ ] 사이드바 폭이 모바일에 맞게 줄어드는가
- [ ] 메인 콘텐츠가 사이드바에 가리지 않는가

---

## 5) Week06 체크

- [ ] 입력폼이 한 화면 안에 자연스럽게 들어오는가
- [ ] TextField 높이가 너무 크지 않은가
- [ ] 버튼이 화면에 맞게 줄어드는가

---

## 6) 참고

- 모바일 화면에서 문제가 있으면 `App.jsx`의 `createTheme` 반응형 설정을 확인하세요.
- `sx`에서 `xs`, `sm` 값을 사용하면 모바일 전용 조정이 가능합니다.
