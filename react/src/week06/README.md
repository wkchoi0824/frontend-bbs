# Week 06: 상태 관리 & 화면 완성

목표:
- 입력 처리와 form 관리
- 상태 관리 흐름 이해
- 글 작성 화면 구성

파일 구성:
- `pages/Week06Page.jsx`: 페이지 연결
- `pages/WritePostPage.jsx`: 글 작성 페이지
- `components/PostForm.jsx`: 입력 폼 컴포넌트
- `hooks/useForm.js`: 입력 상태 관리 훅
- `store/postStore.js`: 간단한 상태 저장소 예시

**코딩 스타일**
- Week06는 **화살표 함수 스타일**로 작성되었습니다.
  - Week05에서 익힌 스타일을 실제 폼/상태 관리에 적용합니다.
  - 실무에서 흔한 작성 방식에 익숙해지도록 구성합니다.
