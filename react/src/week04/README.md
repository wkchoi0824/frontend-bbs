# Week 04: React 기초

목표:
- JSX 기본 구조 이해
- 컴포넌트 분리
- props / state 흐름 이해
- 실습: 게시판 목록 출력

파일 구성:
- `pages/Week04Page.jsx`: 화면 엔트리(주차 페이지)
- `components/BoardList.jsx`: 게시글 리스트 컴포넌트
- `components/BoardItem.jsx`: 게시글 항목 컴포넌트
- `data/posts.js`: 더미 데이터

**코딩 스타일**
- Week04는 **함수 선언식 스타일**로 작성되었습니다.
  - 초보자에게 읽기 쉬운 문법을 먼저 익히기 위한 의도입니다.
  - 실무에서는 화살표 함수가 많지만, 선언식도 여전히 많이 사용됩니다.
